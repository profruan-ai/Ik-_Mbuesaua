# Ik-_Mbuesaua
Iké Mbuesaua é uma ferramenta digital que contextualiza questões escolares para a realidade cultural dos povos indígenas. A plataforma adapta perguntas de materiais didáticos para que estejam alinhadas ao cotidiano, saberes e cultura das comunidades, fortalecendo uma educação escolar indígena mais significativa para crianças nas aldeias.
